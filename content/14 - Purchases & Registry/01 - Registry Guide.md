---

A baby registry is both a practical tool and a gift guidance document. Building a good one requires resisting the urge to add everything you see, and instead identifying what you'll actually use.

---

## When to Start

- **Weeks 12–16:** Begin browsing; start a "considering" list
- **Weeks 16–20:** Build and publish the registry (by anatomy scan; before announcing the sex)
- **Weeks 20–28:** Finalize; send to baby shower host

---

## Where to Register

**Register at 1–2 retailers max.** More creates confusion. Consider:

| Retailer | Pros | Cons |
|---|---|---|
| Amazon | Universal; huge selection; universal address for other brands; completion discount | Quality varies; harder to return |
| Buy Buy Baby / buybuy BABY | Baby-specific; good for trying items; completion discount | Closing many locations |
| Target | Widely accessible; good return policy; 15% completion discount | More limited selection |
| Babylist | Universal registry; links items from any store | Not a single-store discount |

**Recommendation:** Babylist (to aggregate everywhere) + one major retailer for completion discount and easy gifting.

---

## Registry Strategy

### Before You Build
- Research each category before adding; don't just add everything
- Know your stroller before you add anything (carrier compatibility, car seat compatibility matter)
- Know your car seat before finalizing the stroller

### As You Build
- Include a range of price points ($20–$300+)
- Add more than you think you need in lower-priced items (onesies, burp cloths, wipes)
- Don't over-specify for items guests might already own (blankets, bibs)
- Include consumables (diapers in size 1–2, wipes) — guests love gifting these

### After the Shower
- Use the completion discount (typically 10–15%) on remaining items
- Buy essentials before birth; defer "nice to have" items to see if you actually need them

## Registry Categories Checklist

### Sleep
- [ ] Bassinet or mini crib
- [ ] Full-size crib (if using from start or planning ahead)
- [ ] Mattress + 3–4 fitted sheets
- [ ] White noise machine (Hatch Baby Rest+)
- [ ] Swaddle blankets (HALO SleepSack Swaddle + Aden + Anais muslin)
- [ ] Monitor (if separate from nursery; video optional)

### Feeding
- [ ] Nursing pillow (Boppy or My Brest Friend)
- [ ] Nursing bras (2–3)
- [ ] Breast pump (often covered by insurance — check before adding to registry)
- [ ] Bottles (2–4 to start; Philips Avent, Dr. Brown's, Comotomo — baby may have preferences)
- [ ] Bottle brush
- [ ] Drying rack
- [ ] Burp cloths (12+)
- [ ] Nursing cover (if desired)
- [ ] Formula (if formula feeding; brand TBD)

### Diapering
- [ ] Changing pad + cover
- [ ] Diaper caddy
- [ ] Diapers: newborn (1 box); size 1 (2–3 boxes)
- [ ] Wipes (unscented; 4–6 packs to start)
- [ ] Diaper cream (Aquaphor + Desitin or equivalent)
- [ ] Diaper bag (backpack style)

### Bathing
- [ ] Baby bathtub with newborn insert
- [ ] Baby washcloths (6–8)
- [ ] Fragrance-free baby wash/shampoo
- [ ] Hooded towels (3)

### Clothing
- [ ] Onesies: newborn 5–7; 0–3 months 10–12; 3–6 months 6–8
- [ ] Footie pajamas (same quantity)
- [ ] Hats (5–6 newborn size)
- [ ] Socks (10 pairs)
- [ ] Mittens (5–6; prevent scratching)

### Gear
- [ ] Infant car seat (research thoroughly first)
- [ ] Stroller (research thoroughly first)
- [ ] Baby carrier/wrap
- [ ] Bouncer or swing (try before committing to expensive models)
- [ ] Play mat / activity gym

### Health and Safety
- [ ] Rectal thermometer
- [ ] Nasal aspirator (NoseFrida)
- [ ] Nail clippers or file (baby nail care kit)
- [ ] Baby-safe pain reliever (Infant Tylenol — have on hand)
- [ ] Baby monitor
- [ ] Baby-proofing items (outlet covers, cabinet locks — not urgent for newborn stage)
